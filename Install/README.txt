This is the installer for the FEDEM desktop application on the Windows platform.

To install, execute the INSTALL.bat file with administrator privileges,
i.e., on Windows 10, right-click on the INSTALL.bat file in the file explorer
and select "Run as administrator".

It will (by default) install the software in the "C:\Program Files\FEDEM" folder,
set the file association for FEDEM model files to the GUI executable (Fedem.exe),
and create a short-cut to Fedem.exe on the user's Desktop.
